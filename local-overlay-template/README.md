# .local/ Overlay Directory

This directory is the template for your deployment's local secrets and state.
Copy it to `.local/` and populate with real values:

    cp -r local-overlay-template .local
    python scripts/init_local_overlay.py --force

Or run the automated scaffold directly:

    make init-local

## Structure

Each subdirectory corresponds to a service. Files with `.example` suffix
show the expected format — rename them (drop `.example`) and fill in real values.

## Secret Origins

- **generated_by_repo**: Automatically generated by `make init-local`
- **provided_externally**: You must provide these (API keys, registrar tokens, etc.)
- **provided_or_generated_externally**: Generate yourself or obtain from a provider

## Never Commit This Directory

`.local/` is in `.gitignore` and protected by a pre-commit hook.
See ADR 0376 for the full incident analysis.
