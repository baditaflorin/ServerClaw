# Release 0.153.0

- Date: 2026-03-25

## Summary
- implemented ADR 0165 with deterministic workflow idempotency keys, a shared file-backed plus Postgres-backed idempotency store, and scheduler-side duplicate suppression before Windmill re-submission
- exposed `execution.idempotent_hit` in the mutation ledger, added `lv3 intent status <intent_id>` for cached-result visibility, and scoped closure-loop retries by trigger reference so duplicate observation deliveries do not fan out extra live jobs
- added the live Postgres schema migration at `migrations/0016_idempotency_store.sql`, operator runbook coverage, and focused regression tests across scheduler, conflict, closure-loop, and CLI surfaces
