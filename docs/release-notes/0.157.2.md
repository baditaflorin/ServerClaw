# Release 0.157.2

- Date: 2026-03-25

## Summary
- fix the delegated Windmill ADR 0158 config-merge migration so the database-side `psql` step runs as `postgres` instead of `root`
- unblock the current `main` Windmill converge path after the live platform exposed the delegated migration failure during a production apply attempt
- mounted the API gateway config bundle at `/app/config` so platform modules used by the gateway can resolve repo-relative config files such as `config/ledger-event-types.yaml`
- raised `ansible_become_timeout` to `120` seconds for the docker-runtime API gateway and Windmill converges so the standard live-apply path can complete the larger recursive sync steps
- added a regression test that keeps the `/app/config` mount in the committed gateway compose template

## Platform Impact
- repository version advances to `0.157.2`
- platform version remains `0.130.4` until the current `main` live apply is completed and verified
