# Release 0.177.33

- Date: 2026-03-28

## Summary

- accepted ADRs 0224 through 0233 to define a production-oriented
  server-resident operations model, moving more routine control-plane work away
  from the Codex chat client and into mature tools such as Ansible Pull,
  systemd, Windmill, Gitea Actions runners, OPA, OpenBao Agent, Nomad, and
  signed release bundles

## Platform Impact

- no platform version bump; this is a repo-only architecture release that
  defines how the server should increasingly operate from inside the platform
  boundary rather than from a live authoring session
