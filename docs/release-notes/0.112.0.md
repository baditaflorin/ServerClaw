# Release 0.112.0

- Date: 2026-03-24

- implemented ADR 0135 in the repository by adding sensitivity-aware developer-portal generation for ADRs, runbooks, services, reference pages, releases, and API docs
- added page-level sensitivity metadata plus visible sensitivity notices to generated developer-portal pages so operators can distinguish `PUBLIC`, `INTERNAL`, `RESTRICTED`, and `CONFIDENTIAL` material in the published output
- changed the portal render path so `RESTRICTED` ADRs and runbooks publish summary-only pages while `CONFIDENTIAL` documents remain source-only and are omitted from portal indexes
- added targeted docs-site coverage for default sensitivity tagging and restricted-document redaction in `tests/test_docs_site.py`
- recorded ADR 0135 and its workstream metadata on `main`, and updated the developer-portal runbook to document the new metadata contract

Platform impact:

- no live platform version bump; this release changes repository-side portal generation and documentation metadata, and those changes become live on the next docs-portal deployment from `main`
