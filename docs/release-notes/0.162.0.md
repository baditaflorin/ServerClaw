# Release 0.162.0

- Date: 2026-03-26

## Summary
- implement the first ADR 0166 rollout with a shared `config/error-codes.yaml` registry, reusable canonical error helpers, and canonical error envelopes for the API gateway and platform-context API
- harden the `api-gateway` and `platform-context` runtime packaging so fresh worktrees ship the shared registry and helper module into their live-apply bundles
- add operator documentation, workstream tracking, and repository validation coverage for the canonical error registry and response contract

## Platform Impact
- repository version advances to `0.162.0`; platform version remains `0.130.12` until ADR 0166 is live-applied from main.
