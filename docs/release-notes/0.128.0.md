# Release 0.128.0

- Date: 2026-03-25

## Summary
- Implemented ADR 0124 by adding `config/event-taxonomy.yaml`, `config/nats-streams.yaml`, and the shared `platform/events/taxonomy.py` helpers as the canonical platform event contract.
- Validated event routing end-to-end with `scripts/validate_nats_topics.py`, wired that check into repository validation and the local pre-push hook, and aligned `config/control-plane-lanes.json` plus `config/api-publication.json` with the active event plane.
- Canonicalized active publishers onto stable `platform.*` subjects, including observation findings, maintenance windows, world-state refreshes, and mutation-ledger fan-out, and updated the related ADRs, workstreams, and runbooks to reflect the new schema.

## Platform Impact
- repository version advances to `0.128.0`
- platform version remains `0.114.6` until this change is applied live from `main`

## Residual Risk
- downstream subscribers that still expect legacy subjects such as `maintenance.*`, `world_state.refreshed`, `ledger.event_written`, or per-check finding topics must be updated before the live event plane is switched over
- reserved ADR 0125+ topics are documented but not yet emitted by live producers
