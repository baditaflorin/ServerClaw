# Release 0.123.1

- Date: 2026-03-24

## Summary
- Implemented ADR 0140 to harden `grafana.example.com` as an operator-only surface instead of an implicitly public observability endpoint.
- Explicitly disabled Grafana public dashboards, kept the managed login-form recovery path, and kept embedding disabled in the repo-managed Grafana configuration.
- Hardened the NGINX edge publication so unauthenticated callers no longer receive Grafana's external `/api/health` version payload and proxied responses strip `X-Grafana-Version` and `Via`.
- Added repository verification that the public platform dashboard URL redirects unauthenticated callers to `/login` instead of rendering dashboard content.

## Platform Impact
- repository version advances to `0.123.1`
- platform version remains `0.114.6` until this change is applied live from `main`

## Residual Risk
- operator access to Grafana still depends on the current Keycloak OIDC login path and its controller-local client secret material.
- the hardening blocks unauthenticated version disclosure and dashboard viewing, but it does not reduce the internal detail visible to already authenticated Grafana users.
