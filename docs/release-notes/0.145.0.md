# Release 0.145.0

- Date: 2026-03-25

## Summary
- add ADR 0167 to `main` and implement the first live degradation-declaration contract across the service capability catalog, validation tooling, API gateway runtime, and health surfaces
- teach the platform API gateway to degrade deliberately during Keycloak JWKS and NATS publication outages, including cached-JWKS auth continuation, explicit `503` fail-fast behaviour after cache expiry, and a durable local NATS outbox
- document the operator workflow for reviewing declared degradation modes and inspecting active degraded state

## Platform Impact
- repository version advances to `0.145.0`
- platform version remains `0.130.3` until the updated API gateway runtime is converged from this mainline revision and recorded with a production live-apply receipt
