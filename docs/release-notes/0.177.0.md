# Release 0.177.0

- Date: 2026-03-27

## Summary

- implement ADR 0176 with a repo-managed Ansible execution-scope catalog, recursive wrapper resolution, and a scoped runner that derives live target hosts from `ansible-playbook --list-hosts`
- wire the `live-apply-*` entrypoints and dedicated converge playbooks through shard rendering plus matching `--limit` enforcement so host-scoped runs stop carrying the full inventory by default
- add focused resolver and runner coverage in `tests/test_ansible_execution_scopes.py`, keep the broader manifest tests green, and fold the new scope-catalog validation into `scripts/validate_repo.sh data-models`

## Platform Impact

- repository version advances to `0.177.0`; platform version is unchanged because ADR 0176 has not been live-applied from `main` yet
