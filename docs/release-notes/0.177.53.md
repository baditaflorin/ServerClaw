# Release 0.177.53

- Date: 2026-03-28

## Summary
- accepted ADRs 0254 through 0263 to define ServerClaw as a chat-first
  self-hosted assistant product on LV3 using Matrix, mautrix,
  OpenClaw-compatible skill packs, Temporal, n8n, Nextcloud, Playwright,

  product architecture on top of the existing LV3 control plane
