# Release 0.178.7

- Date: 2026-04-05

## Summary
- implements ADR 0346 by establishing a centralized port registry, fixing the Gitea ROOT_URL bug that broke Keycloak SSO login (port 3009 leaked into public URL), adding keycloak_internal_http_port to platform_port_assignments, adding _host_proxy_port collision detection to the generator, and migrating 12 role defaults away from hardcoded platform port numbers

## Platform Impact
- fixes Keycloak SSO login on Gitea (gitea_root_url now resolves to https://git.example.com instead of http://git.example.com:3009)
- eliminates hardcoded port numbers in alertmanager_runtime, directus_runtime, dozzle_runtime, gitea_runtime, keycloak_runtime, mail_platform_runtime, monitoring_vm, plausible_runtime, public_edge_oidc_auth, and secret-rotation playbook
