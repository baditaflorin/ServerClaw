# Release 0.87.0

- Date: 2026-03-23

- completed ADR 0083 in repository automation by adding the pinned Docker-based check-runner images, manifest-driven parallel executor, rebuild helper, and `make` surfaces for building, pushing, and running governed validation containers
- completed ADR 0089 in repository automation by adding the build-server cache role, BuildKit and apt-cacher host configuration, cache warming and maintenance helpers, cache-status tooling, and remote execution hooks for shared pip, Packer, and Ansible cache surfaces
- updated ADR 0083 and ADR 0089 plus their workstream metadata to record repository implementation on `0.87.0`, while leaving the live-apply verification and schedule enablement for a follow-up apply from `main`

Platform impact:

- no direct live platform change is claimed in this release commit; the merged repository automation is ready, but the cache host still needs to be applied live from `main` before warm-cache timing and scheduled-maintenance claims become true on the platform
