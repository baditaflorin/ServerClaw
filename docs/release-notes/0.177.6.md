# Release 0.177.6

- Date: 2026-03-27

## Summary
- Integrated ADR 0101 automated certificate lifecycle management on `main`, including the repo-managed OpenBao and Vaultwarden renewal timers plus the validated `step-ca`-backed TLS probe contract.
- Hardened `step_ca_runtime` live apply with an SSH reconnect guard so the final verification block survives the post-start restart window on `docker-runtime-lv3`.
- Hardened `openbao_runtime` live apply with bounded retries around the flakiest mutation endpoints, and corrected the short-lived `step-ca` proxy probe policy so the certificate catalog no longer reports a built-in false critical.

## Platform Impact
- platform_version advances to 0.130.27; ADR 0101 is now live-applied on main with resilient step-ca and OpenBao replays plus verified OpenBao, Vaultwarden, and step-ca certificate probe coverage.
