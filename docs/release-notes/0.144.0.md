# Release 0.144.0

- Date: 2026-03-25

## Summary
- implemented ADR 0159 in repository automation by extending the conflict registry and budgeted scheduler with opt-in speculative execution, probe-driven commit, and compensating rollback support
- added the controller-side speculative execution surfaces under `platform/scheduler/speculative.py`, `scripts/lv3_cli.py --allow-speculative`, and the workflow-catalog schema validation for the new `speculative` block
- documented the feature in ADR 0159, a dedicated speculative execution runbook, and a registered workstream, with regression coverage for compiler opt-in, speculative conflict registration, commit, rollback, and ledger event emission

## Platform Impact
- no live platform version bump; this release implements ADR 0159 speculative execution in repository automation only
