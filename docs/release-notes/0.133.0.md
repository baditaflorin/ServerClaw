# Release 0.133.0

- Date: 2026-03-25

## Summary
- Implemented ADR 0134 with a config-backed redaction pipeline for deployment history and changelog portal content.
- Added `config/changelog-redaction.yaml` and `scripts/changelog_redaction.py` to mask actor emails, internal hostnames, private IPs, and inline secrets while stripping raw error detail and stack traces.
- Extended deployment-history loading and validation so mutation-audit metadata is summarized safely and focused tests cover both `get-deployment-history` output and rendered portal HTML.

## Platform Impact
- no live platform version bump; this release updates repository automation, release metadata, and operator tooling only
