# Release 0.109.0

- Date: 2026-03-24

- implemented ADR 0102 in repository automation with a dedicated Lynis scan playbook, a report parser, a runtime-image Trivy scanner, and a controller-side security posture aggregator that writes committed receipts under `receipts/security-reports/`
- added the repo-managed `security-posture-scan` Windmill wrapper, `make security-posture-report`, the `platform.security.*` control-plane lane metadata, and optional Mattermost, GlitchTip, NATS, and Influx integration points for weekly security reporting
- extended the current ops portal and managed Grafana platform overview dashboard template with Security Posture summary surfaces sourced from the latest receipt and the `platform_security_posture_*` metrics
- documented the operator workflow in `docs/runbooks/security-posture-reporting.md` and updated ADR 0102 plus the workstream registry to mark the repository implementation as complete in `0.109.0`

Platform impact:

- no live platform version bump; this release adds repository-side security posture automation and status surfaces that still need scheduling and live apply from `main`
