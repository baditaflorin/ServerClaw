# Release 0.158.0

- Date: 2026-03-25

## Summary
- implement ADR 0155 on current `main` by queueing workflow-concurrency and intent-conflict rejections behind a durable scheduler intent queue
- add the repo-managed dispatcher script, Windmill wrapper, and safety schedule used to replay queued intents after releases
- expose ADR 0155 queue controls through `lv3 run` and document the operator path with a dedicated workstream and runbook
- add ADR 0149 Semaphore as a repo-managed private Ansible job console with a PostgreSQL backend on `postgres-lv3`, a runtime stack on `docker-runtime-lv3`, and a Tailscale-only host proxy on `proxmox_florin`
- seed the `LV3 Semaphore` project, localhost inventory, API token flow, and `Semaphore Self-Test` template through repo-managed bootstrap code instead of UI-only state
- harden the Semaphore runtime checkout, compose recreation, and API client status-code handling so the live task runner can execute the seeded self-test reliably after rebases and reruns
- record ADR 0149 as implemented in repository version `0.158.0` and platform version `0.130.7` after the integrated mainline live apply

## Platform Impact
- repository version advances to `0.158.0`
- platform version advances to `0.130.7` because the 2026-03-25 mainline live applies verified both the integrated Semaphore converge rerun and the ADR 0155 Windmill queued-intent dispatcher rollout
