# Release 0.124.1

- Date: 2026-03-24

## Summary
- Fixed the Grafana dashboard Jinja templates used by ADR 0140 so repo-managed dashboard imports render JSON `null` values correctly during live converge instead of failing with `AnsibleUndefinedVariable`.
- Fixed the shared `make preflight` wrapper to run `preflight_controller_local.py` with `PyYAML`, restoring the repository-first `make configure-edge-publication` and related workflows.
- Kept ADR 0140's public-surface hardening intact while removing the two rollout blockers uncovered during the first live-apply attempt.

## Platform Impact
- repository version advances to `0.124.1`
- platform version remains `0.114.6` until this change is applied live from `main`

## Residual Risk
- edge publication still depends on locally generated static portal artifacts under `build/changelog-portal/` and `build/docs-portal/` being present on the controller before rollout.
- operator access to Grafana still depends on the current Keycloak OIDC login path and its controller-local client secret material.
