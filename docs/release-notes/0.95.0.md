# Release 0.95.0

- Date: 2026-03-23

- implemented ADR 0088 with repo-managed ephemeral fixture definitions, a reusable `tofu/modules/proxmox-fixture` wrapper, the `scripts/fixture_manager.py` lifecycle helper, and the `scripts/vmid_allocator.py` Proxmox VMID allocator
- added `make fixture-up`, `make fixture-down`, and `make fixture-list`, plus the Windmill `config/windmill/scripts/fixture-expiry-reaper.py` workflow and ignored runtime receipts under `receipts/fixtures/`
- added the delegated Molecule `proxmox-fixture` driver together with a demo `docker_runtime` scenario that provisions a real VM fixture and hands the inventory off to Molecule
- documented the operator flow in `docs/runbooks/ephemeral-fixtures.md` and updated ADR 0088 plus the workstream registry to mark the repository implementation as complete in `0.95.0`

Platform impact:

- no live platform version bump; fixture live rollout still depends on ADR 0072's staging bridge being present on the host before `vmbr20` fixtures and the scheduled reaper can be enabled from `main`
