# Release 0.127.0

- Date: 2026-03-25

## Summary
- Implemented ADR 0139 with a deterministic `config/subdomain-exposure-registry.json` generated from the governed subdomain catalog and repo-managed edge publication surfaces.
- Added `scripts/subdomain_exposure_audit.py`, the worker wrapper `config/windmill/scripts/subdomain-exposure-audit.py`, and the operator runbook `docs/runbooks/subdomain-exposure-audit.md`.
- Extended `config/subdomain-catalog.json` with explicit `auth_requirement` values and corrected the production status of `ops.lv3.org`, `docs.lv3.org`, and `changelog.lv3.org` to match live DNS.

## Platform Impact
- repository version advances to `0.127.0`
- platform version remains `0.114.6` until this change is applied live from `main`

## Residual Risk
- the repository now detects exposure drift, but the weekly worker schedule and any downstream notifications still require live apply from `main`
- some public portals remain intentionally unauthenticated in the catalog until their separate hardening ADRs are applied
