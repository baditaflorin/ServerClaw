# Release 0.151.0

- Date: 2026-03-25

## Summary
- implemented ADR 0157 with per-VM execution-lane budgets, workflow resource reservations, and TTL-backed scheduler admission control for mutation workflows
- added the execution-lane catalog, validation tooling, operator runbook, and Windmill live-apply workstream documentation for the lane-budget model
- Implement ADR 0144 by moving the management mesh control plane to repo-managed Headscale, publishing `headscale.example.com`, and updating canonical host access to `100.64.0.1`.
- Implement ADR 0162 with a file-backed lock registry, coordination map, intent queue, and Windmill deadlock detector runtime; live apply remains pending while the Proxmox host is unreachable.

## Platform Impact
- no live platform version bump yet; this release merges ADR 0157 lane budgets and resource reservations into main before the first verified Windmill live apply
