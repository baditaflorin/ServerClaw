# Release 0.83.0

- Date: 2026-03-23

- completed ADR 0077 in repository automation by replacing compose-directory secret env files with OpenBao Agent sidecars that render runtime env files under `/run/lv3-secrets/<service>/runtime.env`
- added a shared OpenBao Agent helper plus per-service AppRole contracts for Windmill, Keycloak, Mattermost, Open WebUI, NetBox, the private platform-context API, and the mail-platform gateway
- added a repository validation gate that fails if any `*.env` file exists in the checkout and documented the runtime-secret injection operating model in a dedicated runbook
- updated ADR 0077 and its workstream metadata to record repository implementation on `0.83.0`

Platform impact:

- no direct live platform change is claimed in this release commit; the repo-managed migration is ready for a deliberate apply from `main`
