# Release 0.177.109

- Date: 2026-03-30

## Summary
- implements ADR 0293 by adding the private Temporal durable workflow engine, PostgreSQL-backed history and visibility stores, OpenBao-managed runtime secrets, and the repo-managed `lv3` namespace, and ADR 0288 by adding the repo-managed Flagsmith runtime, OpenBao-backed seed and environment key publication, public `flags.lv3.org` edge routing, and live verification hardening for sparse feature-state API responses plus Docker bridge-chain recovery

## Platform Impact
- advances platform_version to `0.130.72` with the Temporal exact-main replay and preserves that verified baseline while integrating the public Flagsmith service behind the shared oauth2-proxy boundary on `flags.lv3.org`
