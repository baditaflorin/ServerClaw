# Release 0.92.0

- Date: 2026-03-23

- implemented ADR 0085 with a reusable `tofu/modules/proxmox-vm` module plus `production` and `staging` OpenTofu environments
- added the governed OpenTofu entry points `make remote-tofu-plan`, `make remote-tofu-apply`, `make tofu-drift`, and `make tofu-import`
- added containerized OpenTofu execution wrappers that run on `build-lv3`, including host-networked remote execution for reliable Proxmox API access
- imported all six live production VMs (`110`, `120`, `130`, `140`, `150`, `160`) into OpenTofu state and verified `make tofu-drift ENV=production` returns `No changes`
- documented the production import and day-to-day lifecycle flow in `docs/runbooks/tofu-vm-import.md` and `docs/runbooks/tofu-vm-lifecycle.md`
- updated ADR 0085 and the workstream registry to mark the OpenTofu VM lifecycle as implemented in repo version `0.92.0`

Platform impact:

- no platform version bump; this release adds repository-managed OpenTofu VM lifecycle automation and verifies it against the current live Proxmox guests
