# Release 0.124.0

- Date: 2026-03-24

## Summary
- implemented ADR 0135 by making the developer-portal generator sensitivity-aware for ADRs, runbooks, service/reference pages, releases, and API documentation
- added page-level sensitivity metadata and visible notices to generated docs pages, with `RESTRICTED` documents rendered as summary-only and `CONFIDENTIAL` documents kept source-only
- added targeted docs-site coverage for default sensitivity tagging and restricted-document redaction, and documented the metadata contract in the developer-portal runbook

## Platform Impact
- no live platform version bump; this release updates repository automation, release metadata, and operator tooling only
