# Release 0.159.0

- Date: 2026-03-25

## Summary
- finalize the first live apply for ADR 0170 on current `main` by converging the API gateway and Windmill with the shared timeout hierarchy runtime paths active
- harden the API gateway runtime packaging so the live container receives the repo `platform/`, `scripts/`, and `config/` trees through staged archive sync plus a config-bind probe and forced recreate path
- repair the OpenBao dependency chain used by Windmill and other runtime secret injection flows by targeting the PostgreSQL primary over the private guest address, fixing dynamic-credential template placeholders, splitting late secret-rotation facts, and auto-unsealing the local OpenBao API when it restarts sealed
- record ADR 0170 as live on repository version `0.159.0` and platform version `0.130.8`

## Platform Impact
- repository version advances to `0.159.0`
- platform version advances to `0.130.8` because the 2026-03-25 mainline live apply verified the timeout hierarchy on the API gateway and Windmill control-plane paths, while the dependent OpenBao runtime verification passed even though `postgres-replica-lv3` stayed SSH-unreachable during the OpenBao prep play
