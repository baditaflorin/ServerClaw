# Release 0.148.0

- Date: 2026-03-25

## Summary
- Implemented ADR 0151 with repo-managed `n8n` on `docker-runtime`, PostgreSQL persistence on `postgres`, edge publication at `https://n8n.example.com`, and public webhook path handling under the shared ingress model.
