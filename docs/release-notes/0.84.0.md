# Release 0.84.0

- Date: 2026-03-23

- completed ADR 0080 in repository automation by adding the NATS KV-backed maintenance-window controller, schema, governed workflows and command catalog entries, Windmill wrapper, and maintenance-window observe tool surface
- wired the ADR 0071 observation loop to read active maintenance windows, mark planned non-security findings as `suppressed`, and omit Mattermost or GlitchTip forwarding while preserving structured outputs and NATS publication
- updated ADR 0080 and its workstream metadata to record repository implementation on `0.84.0`, while documenting the remaining live NATS publish-permission blocker for opening and closing windows from the controller

Platform impact:

- no direct live platform change is claimed in this release commit; the merged repository automation is ready, but the live NATS principal set still needs publish rights for `$KV.maintenance-windows.>` before controller-managed windows can be applied on the platform
