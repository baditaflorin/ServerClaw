# Release 0.85.0

- Date: 2026-03-23

- completed ADR 0086 in repository automation by moving the Ansible source of truth into `collections/ansible_collections/lv3/platform/` and keeping repo-root compatibility symlinks for `roles/`, `filter_plugins/`, and `callback_plugins/`
- rewrote repo playbooks and role-to-role imports to use `lv3.platform.*` FQCNs so the same automation runs both from the repo checkout and from a built collection artifact
- added collection metadata, `make collection-sync`, `make collection-build`, `make collection-install`, `make collection-publish`, and the Windmill `config/windmill/scripts/collection-publish.py` helper for internal Galaxy publication
- added the shared `preflight`, `common_handlers`, `secret_fact`, and `wait_for_healthy` roles, wired `preflight` into six existing roles, and documented the packaging workflow in `docs/runbooks/ansible-collection-development.md`
- updated ADR 0086 and its workstream registry/status metadata to record implementation on `0.85.0`

Platform impact:

- no direct live platform change is claimed in this release commit; the collection packaging and publication workflow is now ready from `main`
