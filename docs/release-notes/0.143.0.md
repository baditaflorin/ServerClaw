# Release 0.143.0

- Date: 2026-03-25

## Summary
- implement ADR 0125 with a repo-managed `config/agent-policies.yaml` registry, the new `platform.agent_policy` enforcement engine, and bounded autonomous execution checks in both the goal compiler and scheduler
- extend `lv3 run` with actor-aware autonomous submission controls, validate agent policy data models in repository checks, and document the operator workflow for capability-bound denials and escalations
- add targeted tests that prove out-of-bounds autonomous actions are denied or escalated before execution, including workflow-tag, risk-ceiling, and daily-cap enforcement paths

## Platform Impact
- repository version advances to 0.143.0; platform version remains 0.130.2 because ADR 0125 is repo-integrated only and has not been live-applied from `main`
