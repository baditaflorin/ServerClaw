# Release 0.177.145

- Date: 2026-04-02

## Summary
- completes the repo-managed runtime-pool transition program by classifying all services, implementing the `runtime-general` and `runtime-control` deployment surfaces, and adding governed runtime-pool memory-envelope and autoscaling contracts

## Platform Impact
- Repo-only change. No live platform apply was performed in this release.
- `runtime-control-lv3` is now represented in inventory, lane ownership, control-plane service catalogs, and runbooks as the target control-plane pool.
- ADR 0321 and ADR 0322 are now implemented in repository automation and validation, with live enforcement still pending a future guarded mainline apply.
