# Release 0.157.0

- Date: 2026-03-25

## Summary
- implemented ADR 0170 with a repository-managed timeout hierarchy, deadline propagation helpers, repo validation, and runtime timeout integration across the API gateway, scheduler, world-state workers, drift helpers, and NetBox sync
- normalized API gateway upstream budgets to the ADR 0170 `http_request` default and bundled `config/timeout-hierarchy.yaml` into the live gateway runtime payload
- aligned the timeout hierarchy docs and scheduler runbooks with the existing ADR 0172 watchdog path and the newer ADR 0157 execution-lane model

## Platform Impact
- no live platform version bump yet; this release merges ADR 0170 into `main` before the first verified live apply because the Proxmox host and published API path were unreachable during the integration window
