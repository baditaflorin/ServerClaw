# LV3 Platform Docs

This site is the generated developer portal for the LV3 platform. It is built from the canonical repository catalogs, ADR corpus, release notes, and runbooks so operators and integrators can answer practical questions without chasing configuration across the repo.

Use it for:

- service URLs, ports, exposure, and owning ADRs
- runbook discovery for routine operations
- architecture decisions and implementation status
- release notes, changelog context, and upgrade guidance
- reference tables for subdomains, identities, and secret contracts

The live status surface remains the operations portal and monitoring stack. This documentation site is the durable, version-controlled reference layer.
